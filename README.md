## 🚀 Quick Start

### Option 1 — Open directly
```bash
# Clone or download the repo, then:
open gcp_platform_architect_prep.html       # macOS
start gcp_platform_architect_prep.html      # Windows
xdg-open gcp_platform_architect_prep.html   # Linux
```

### Option 2 — Serve locally (if browser blocks local file fonts)
```bash
# Python 3
python3 -m http.server 8080
# Then open: http://localhost:8080/gcp_platform_architect_prep.html

# Node.js (npx)
npx serve .
```

### Option 3 — Host on GitHub Pages
1. Fork this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Access at `https://<your-username>.github.io/<repo-name>/gcp_platform_architect_prep.html`

## 🛠️ Technical Implementation

```
Architecture:
├── Fixed sidebar (230px)          → Section navigation
├── Sticky topbar                  → Current section title + Q&A count
├── Content panels (display:none)  → show() toggles active panel
├── Q&A cards (.qc)                → tQ() toggles expand/collapse
├── Filter buttons (.fb)           → filterQ() shows/hides by data-cat
└── Study Guide sub-tabs (.sg-tab) → sgShow() toggles sub-panel

State management: Pure DOM — no frameworks, no build tools required

CSS variables:
  --navy     #080D1A   Background
  --card     #192035   Card background
  --blue     #4A8FE2   Primary accent
  --green    #35B870   Skills Boost / success
  --amber    #E0A040   Warning / intermediate
  --red      #D85858   Error / gap
```

### Recommended study flow

```
Week 1 — Gap areas first (highest ROI before interview)
  Day 1-2  →  📚 Study Guide: Vertex AI & LLMs tab
              Read Vertex AI Agent Builder docs
              Complete "Introduction to Generative AI" Skills Boost
  Day 3    →  🤖 Vertex AI Q&As section

  Day 7    →  🎯 Behavioural Leadership (all 20 questions)
              🏢 Enterprise Architecture STAR scenarios (filter: 🎯 STAR Scenarios)

Week 2 — Reinforce strengths + mock interview
  Day 1-2  →  🔧 Terraform + ☁️ Senior GCP Architecture
```

### Tips for using the Q&As

- **Filter first**: use category filter buttons to focus on specific sub-topics
- **STAR scenarios**: filter the Enterprise Architecture section by "🎯 STAR Scenarios" for scenario practice
- **Difficulty progression**: work Core → Intermediate → Senior within each section
- **Senior questions**: these are the differentiating questions — spend the most time here
- **`[USE YOUR EXPERIENCE]`**: sections marked this way in answers are prompts to substitute your own real examples
